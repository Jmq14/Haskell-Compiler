# Haskell-Compiler

First install stack.

Second, run

```sh
tools/setup.sh
```

or

```sh
python tools/setup.py
```

to setup.

To compile, run

```sh
tools/compile.sh
```

or

```sh
python tools/compile.py
```

To run code, run

```sh
tools/run.sh
```

or

```sh
python tools/run.py
```

To test, run

```
tools/test.sh
```

or

```sh
python tools/test.py
```

(If test is not available, run 

```sh
stack install QuickCheck
```
