# Haskell-Compiler

First install stack.(http://www.baidu.com)

Second, run

```sh
tools/setup.sh
```

or

```sh
python tools/setup.py
```

to setup.

To compile, run

```sh
tools/compile.sh
```

or

```sh
python tools/compile.py
```

To run code, run

```sh
tools/run.sh
```

or

```sh
python tools/run.py
```
